HTML Document Structure

<!DOCTYPE html>

Declares the document type and HTML version

<html>

Root element that contains all other HTML elements

<head>

Contains meta-information about the document

<title>

Defines the title of the document (shown in browser tab)

<meta>

Provides metadata about the HTML document

<link>

Links to external resources like CSS files

<script>

Includes JavaScript code or links to JS files

</head>
<body>

Contains the visible content of the document

<header>

Represents introductory content, often contains navigation

</header>
<main>

Contains the main content of the document

</main>
<footer>

Represents footer content for the document or section

</footer>
</body>
</html>
