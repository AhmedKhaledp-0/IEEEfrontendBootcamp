Bootcamp Summary

This 3-day bootcamp is designed to take you from zero to creating your first interactive web game. You'll learn the three core technologies of front-end web development:

  • HTML for structure and content
  • CSS for styling and layout
  • JavaScript for interactivity and logic

By the end, you'll have built a fully functional "Guess the Number" game that demonstrates these technologies working together.
